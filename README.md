# personal-helper

A simple Python-based personal assistant for managing **contacts** and **notes**. Supports CRUD operations, search, tagging, and data persistence using JSON files.  

---

## Features

### Contacts
- Add, edit, delete, and search contacts
- Validate phone numbers and emails
- Track birthdays and upcoming birthdays
- Save and load contacts to/from JSON

### Notes
- Add, edit, delete, and search notes
- Add and remove tags for notes
- Sort notes by tag
- Save and load notes to/from JSON

### Utilities
- Validation for phone numbers and email addresses
- Custom exceptions for better error handling
- Simple storage service for persistent data
